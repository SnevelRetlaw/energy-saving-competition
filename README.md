# Energy-saving-competition
Repository for the energy-saving competition
It contains 4 folders, each designated to a specific part of the competition.

## Competition content
This folder contains the information on challenges and the Daily Energy Insights as they have been used in the competition

## Supabase content
This folder contains the schema of the database as well as the code for the different edge functions that have been used in the competition

## Webapp content
This folder contains all code that is used for the webapp. It is a simple static html page.

## Transcription
This folder contains the script that was used to transcribe the interviews. Careful! extremely complicated!
